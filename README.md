# Game-Library-Manager---Projeto-Acad-mico
O Game Library Manager é uma aplicação web para gerenciamento de uma biblioteca de jogos físicos, permitindo o cadastro de usuários, jogos e o controle de empréstimos.  O sistema foi projetado seguindo uma arquitetura em camadas (Controller, Service, Repository, Model), utilizando princípios REST e separação de responsabilidades.
